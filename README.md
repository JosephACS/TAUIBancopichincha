##  Descripción

Diseño **Responsivo del Inicio de Sesión** de una Aplicación Móvil inspirada en la app oficial del **Banco Pichincha**, una de las entidades bancarias más importantes del Ecuador.

Este proyecto fue desarrollado como parte de una actividad académica con el objetivo de replicar fielmente la interfaz visual de la pantalla de login de la Banca Móvil del Banco Pichincha, utilizando **XML tradicional** en Android Studio.

---

##  Objetivo

Elaborar el Diseño Responsivo del Inicio de Sesión de una Aplicación Móvil de una empresa popular del país, tomando como referencia la **App Móvil del Banco Pichincha versión 9.5.0**.

| Tecnología | Versión | Descripción |

| XML | - | Diseño de layouts |
| Material Design 3 | 1.11.0 | Componentes visuales |
| Minimum SDK | API 24 (Android 7.0) | Compatibilidad del 99.2% de dispositivos |
| Target SDK | API 34 (Android 14) | SDK objetivo |
